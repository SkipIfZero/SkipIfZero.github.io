About

SkipIfZero is a one-man organisation dedicated to creating games and other software. It was founded back in 2012 when I was creating my first game, Space Hunter, as part of a school project.

The name SkipIfZero comes from basic control flow of a program. An if statement skips its content if the argument evaluates to 0. It was also the name of an assembly instruction in a microprocessor I used to write code for.

The long term goal is to expand and create bigger, more interesting games. Maybe even make a profit. If you're interested in joining or collaborating, contact me.
